Resource Center

Redesigning resource center page of pnypartners.com

Issue

In the previous design, people had hard time finding the files they were looking for.

Criteria
  • Target User: PNY Partners, PNY Sales Team.
  • Availability: Filters were possible to be added.
  • Things to keep in mind:
    Making sure there is a visible difference between category and file.
    Making them simple and obvious.
  • Reference:Document Management System (Ex: Sharepoint, Hubspot)
File Organization

There were many different types of files but mainly two ways of organizing files were possible: by the type of asset and type of product it was about.


Wireframe

We created multiple examples of wireframes and narrowed down to two directions: One with the filters and other with the combination of sidebar.I focused on the combination of sidebar.


Prototypes
  • After review, the wireframe with the sidebar received most positive comments, so we developed prototype based on it.
  • Narrowed down to three main designs using slightly different color and different styles of filters.
  • Started thinking about how it works on responsive design - especially in mobile.

Final Prototypes
  • Category based on asset type
  • Filters based on the product type
  • Filters based on file type
  • Can be sorted by ID, type and title
  • Download icon shows up when hover
